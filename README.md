# Retro Platform Test 🐾

Proyecto personal — plataformas 2D retro.

## Cómo jugar

Abrir `index.html` con **Live Server** en VS Code.

### Controles
- `← →` o `A D` — mover
- `ESPACIO` o `↑` — saltar
- Doble `ESPACIO` — doble salto
- Saltar **encima** de enemigos para eliminarlos

## Archivos de audio

Coloca tus archivos de música aquí:
- `assets/audio/bgm.mp3` — música del nivel
- `assets/audio/end_theme.mp3` — música de la escena final

> Si no hay archivos de audio, el juego funciona igual (sin sonido).

## Subir a GitHub Pages

1. Sube todo el proyecto a un repositorio en GitHub
2. Ve a **Settings → Pages**
3. Selecciona la rama `main` y carpeta `/root`
4. Listo — tendrás una URL para compartir

## Estructura

```
retro-platform-test/
├── index.html
├── style.css
├── game.js
├── assets/
│   └── audio/
│       ├── bgm.mp3
│       └── end_theme.mp3

```
